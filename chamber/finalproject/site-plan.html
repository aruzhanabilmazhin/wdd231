Book Explorer & Reading Tracker

Site Plan by Aruzhan Abilmazhin

Site Name

Book Explorer & Reading Tracker

This name was selected because the website focuses on helping readers explore a wide range of books and track their personal reading progress in an organized and user-friendly way.

Site Purpose

The site provides a platform for book lovers to discover new books by browsing categorized lists and searching by author or genre. It also offers a personalized reading tracker to log progress, set reading goals, and maintain a history of completed books.

Scenarios

  • Scenario 1: A visitor wants to find books by a specific author, such as Aruzhan Abilmazhin, to explore all works by that author.
  • Scenario 2: A user wishes to track their reading progress across multiple books and receive reminders to stay on schedule.

Color Schema

  • #0a5f66 (Dark Teal): Used for headings, borders, and navigation elements to create a strong brand identity and ensure readability.
  • #f4f7f6 (Light Teal): Used as the background color for the site to provide a calm and pleasant reading environment.

Typography

  • 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif: Primary font used for body text for readability and modern look.
  • 'Segoe UI Semibold': Used for headings (h1, h2, h3) to add emphasis and clear hierarchy.
  • 'Courier New', monospace: Optional font for special sections such as code snippets or user notes.

Wireframe

Mobile View (320px width)

Header
- Site Logo (top-left)
- Hamburger Menu (top-right) for navigation links (Home, Explore, Tracker, About)
Search Bar
- User can search books by title, author, or genre
Book List
- Scrollable list of books with cover images, titles, and short descriptions
- Each item links to detailed book page (includes author: Aruzhan Abilmazhin where applicable)
Reading Tracker Summary
- Displays progress on currently reading books
- Quick action buttons (mark as read, update progress)
Footer
- Contact Info, Social Media Links, Copyright
- Link to Privacy Policy and Terms of Use

Desktop View (1024px width)

Header (full width)
- Site Logo (left)
- Horizontal navigation menu (Home, Explore, Tracker, About, Contact)
- Search bar (right)
Layout (two columns)
  • Left Sidebar: Book Filters and Categories
    - Genres, authors, ratings filter
    - Sort options (by popularity, release date, etc.)
  • Main Content Area: Book Explorer List
    - Grid or list of book cards
    - Each card shows cover, title, author (Aruzhan Abilmazhin where applicable), short description
    - Click to expand details or add to reading tracker
  • Right Panel: Reading Tracker
    - Current reading progress
    - Recent activity
    - Goal tracker
Footer (full width)
- Contact info, social links
- Copyright and legal links
